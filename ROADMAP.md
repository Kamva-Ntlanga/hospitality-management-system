# HotelHub Roadmap

## Completed Features (MVP)

- [x] Domain model and class diagram
- [x] Repository pattern with in-memory storage
- [x] Service layer with business logic
- [x] REST API with FastAPI
- [x] CI/CD pipeline with GitHub Actions
- [x] Branch protection and status checks

## Short-Term (Next 2-3 Months)

| Feature | Description | Priority |
|---------|-------------|----------|
| Database Integration | Replace in-memory storage with PostgreSQL | High |
| Authentication | JWT-based user authentication | High |
| Room Photos | Upload and store room images | Medium |
| Email Notifications | Send booking confirmations | Medium |

## Medium-Term (3-6 Months)

| Feature | Description | Priority |
|---------|-------------|----------|
| Payment Gateway | Integrate Stripe for online payments | High |
| Guest Reviews | Allow guests to leave reviews after stay | Medium |
| Reporting Dashboard | Advanced analytics with charts | Medium |

## Long-Term (6-12 Months)

| Feature | Description | Priority |
|---------|-------------|----------|
| OTA Integration | Sync with Booking.com and Expedia | Medium |
| Real-Time Chat | Guest-staff messaging system | Low |
| AI Pricing | Dynamic pricing based on demand | Low |

## How to Contribute

If you're interested in working on any of these features, open an issue and comment on it.
